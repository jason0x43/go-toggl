go-toggl
========

A Go library for accessing the Toggl API

Installation
------------

Include `github.com/jason0x43/go-toggl` as a dependency in your code. The library imports as `toggl`.

Documentation
-------------

See the [godocs](http://godoc.org/github.com/jason0x43/go-toggl).
